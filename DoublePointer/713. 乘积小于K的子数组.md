注意内层`while`循环的索引越界判断要有等号。


```java
public int numSubarrayProductLessThanK(int[] nums, int k) {

    int n = nums.length;

    int m = 1;

    int count = 0;

    int lo = 0, hi = 0;

    while (hi < n) {

        m = m * nums[hi];

        while (m >= k && lo <= hi) {

            m = m / nums[lo];

            lo++;

        }

        count += hi - lo + 1;

        hi++;

    }

    return count;

}
```
